## Подготовка

Перед началом выполнения задачи, пожалуйста, ознакомьтесь с следующими документами для восстановления контекста:
- [Концепция проекта](../docs/bizlevel-concept.md)
- [План разработки](../docs/development-plan.md)
- [Текущий статус проекта](../docs/status.md)

## Описание задачи

Необходимо реализовать полноценную страницу профиля пользователя в приложении BizLevel, которая будет отображать информацию о пользователе, прогресс обучения по всем уровням, а также предоставлять возможность редактирования персональных данных.

### Шаги выполнения:

1. **Создание компонентов для отображения информации о пользователе:**
   - Разработать компонент UserProfile для страницы профиля
   - Реализовать отображение основной информации (имя, email, фото профиля, дата регистрации)
   - Добавить блок с общей статистикой (количество пройденных уровней, тестов, скачанных артефактов)
   - Настроить адаптивный дизайн для разных устройств

2. **Реализация отображения прогресса по уровням:**
   - Создать компонент для визуализации общего прогресса обучения
   - Реализовать компонент для отображения детализированного прогресса по каждому уровню
   - Добавить возможность фильтрации или сортировки уровней
   - Реализовать визуализацию достижений пользователя (бейджи, награды)

3. **Добавление возможности редактирования профиля:**
   - Создать форму для редактирования персональных данных
   - Реализовать функционал обновления информации в базе данных
   - Добавить валидацию полей ввода
   - Реализовать загрузку и обновление аватара пользователя

4. **Реализация загрузки и отображения аватара:**
   - Интегрировать загрузку изображений через Supabase Storage
   - Реализовать кроппинг и оптимизацию загружаемых изображений
   - Добавить предпросмотр аватара перед загрузкой
   - Настроить отображение аватара в различных местах приложения

## Структура страницы профиля

Рекомендуемая структура страницы профиля:
1. **Верхняя секция**:
   - Аватар пользователя
   - Имя и основная информация
   - Общая статистика и достижения
   - Кнопка редактирования профиля

2. **Секция прогресса**:
   - Общий индикатор прогресса
   - Список пройденных/доступных уровней с индикацией прогресса
   - Фильтры для отображения (все, в процессе, завершенные)

3. **Секция настроек**:
   - Форма редактирования профиля (в модальном окне или отдельной вкладке)
   - Настройки уведомлений (если применимо)
   - Управление аккаунтом

## Проверка работы

Для проверки корректности реализации страницы профиля выполните:
- Тестирование отображения всех данных пользователя
- Проверка корректности визуализации прогресса
- Тестирование функционала редактирования профиля и валидации
- Проверка загрузки и отображения аватара
- Тестирование адаптивности на разных устройствах

## Ожидаемый результат

- Полностью функциональная страница профиля с информацией о пользователе
- Визуализация прогресса обучения по всем уровням
- Форма редактирования профиля с валидацией
- Функционал загрузки и отображения аватара
